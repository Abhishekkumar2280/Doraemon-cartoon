# 🧸 Toy Shape using HTML & CSS

A creative cartoon toy character built entirely using **pure HTML and CSS**.

This project demonstrates how complex cartoon-style shapes can be created without using images, SVG, or JavaScript. Everything is designed using div elements and advanced CSS techniques.

---

## 📌 About The Project

This toy character is created using:

- HTML div elements
- Absolute positioning
- Border and border-radius
- CSS transform (rotate)
- Layering using z-index
- Pseudo-elements (::before and ::after)

It is a CSS Art project focused on creativity and layout control.

---

## 🚀 Features

✔ 100% Pure HTML & CSS  
✔ No Images Used  
✔ No JavaScript  
✔ Fully Custom CSS Design  
✔ Uses Pseudo-elements  
✔ Uses Transform & Rotation  
✔ Creative CSS Art
